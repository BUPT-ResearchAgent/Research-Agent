# 🤖 AI智能生成大作业功能使用说明

## 功能概述

SmartEdu智能教育系统现已支持基于课程知识库的AI智能生成大作业功能。该功能能够根据上传的课程资料自动生成综合性、实践性的大作业题目，极大提升了教师的出题效率。

## 🌟 核心特性

### 1. 知识库驱动的AI生成
- 基于RAG（检索增强生成）技术
- 自动从课程知识库中检索相关内容
- 生成与课程内容高度相关的大作业题目

### 2. 综合性题目设计
- 整合多个知识点，避免简单的知识复述
- 联系实际应用场景或案例
- 要求学生运用所学知识解决实际问题

### 3. 智能内容生成
- **详细题目描述**：包括背景、要求、约束条件
- **分层作业要求**：基础要求 + 拓展要求
- **科学评分标准**：内容完整性、分析深度、逻辑清晰、创新性、格式规范
- **解题思路指导**：提供方向性指导，不给出完整答案

## 🚀 使用方法

### 步骤1：准备知识库
1. 在教师端进入"备课与设计" → "构建课程知识库"
2. 上传课程相关资料（PPT、PDF、Word等）
3. 系统自动处理并构建向量知识库

### 步骤2：生成大作业
1. 进入"考核内容生成" → "生成测评"
2. 选择课程和测评名称
3. 在题型选择中勾选"📋 大作业"
4. 系统会显示AI智能生成提示信息：
   - 基于课程知识库，自动生成综合性、实践性的大作业题目
   - 包含详细作业要求、评分标准、解题思路等完整内容

### 步骤3：配置参数
- **题目数量**：1-5题（建议1-2题）
- **分值设置**：每题20-100分
- **时间安排**：根据作业复杂度合理安排
- **特殊要求**：可添加针对性要求

### 步骤4：生成和预览
1. 点击"生成试卷"按钮
2. 系统使用AI智能生成大作业内容
3. 预览生成的大作业，确认内容质量
4. 根据需要进行调整和完善

## 📝 生成内容示例

### 示例大作业格式
```
### 大作业1（50分）

**题目内容**：
基于《数据结构与算法》课程所学知识，设计并实现一个高效的学生信息管理系统...

**作业要求**：
1. 系统需要支持学生信息的增删改查操作
2. 实现至少两种不同的数据结构进行性能对比
3. 提供完整的算法复杂度分析

**拓展要求**：
1. 实现数据的持久化存储
2. 添加用户友好的图形界面

**提交方式**：文档上传（PDF、Word等格式）或文本输入

**评分标准**：
- 内容完整性（30%）：是否完整回答了所有要求
- 分析深度（25%）：分析是否深入透彻
- 逻辑清晰（20%）：论述是否逻辑清晰
- 创新性（15%）：是否有独特见解或创新点
- 格式规范（10%）：文档格式是否规范

**答案**：[参考解答思路和关键要点]
**解析**：[设计意图、考核目标、重点难点分析]
**知识点**：数据结构,算法设计,复杂度分析,系统设计
```

## 🔧 技术架构

### RAG技术流程
1. **文档预处理**：对上传的课程资料进行文本提取和切块
2. **向量化存储**：使用BGE模型将文本转换为向量并存储到Milvus向量数据库
3. **智能检索**：根据题目要求搜索相关知识块
4. **内容生成**：使用DeepSeek大语言模型基于检索内容生成大作业

### 核心组件
- **VectorDatabaseService**：向量数据库服务
- **KnowledgeBaseService**：知识库管理服务
- **DeepSeekService**：AI生成服务
- **ExamService**：考试管理服务
