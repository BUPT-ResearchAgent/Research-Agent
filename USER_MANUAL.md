## 1. 平台简介

SmartEdu 是一套集教师端、学生端、管理端于一体的智能教育平台，支持智能备课、自动出题、在线考试、学情分析、AI问答、合规管理等功能。平台采用网页端操作，支持多角色登录，适用于高校、培训机构等多种场景。

## 2. 系统部署与环境配置

本节介绍如何在本地或服务器上部署 SmartEdu 平台，包括环境准备、Docker与Milvus向量数据库部署、后端启动和前端访问。

### 2.1 环境准备

- 操作系统：建议 Windows 10/11、Linux 或 macOS
- Java 版本：17 及以上
- Maven 版本：3.6 及以上
- 内存建议：至少 4GB 可用内存

### 2.2 安装 Docker

1. 访问 Docker 官网，下载并安装 Docker Desktop。
2. 安装完成后启动 Docker。
3. 在命令行输入 `docker -v` 验证 Docker 是否安装成功。

### 2.3 部署 Milvus 向量数据库

1. 打开命令行，进入你希望存放 Milvus 配置的目录。
2. 拉取 Milvus 官方 Docker Compose 配置文件：

   ```bash
   curl -fsSL https://raw.githubusercontent.com/milvus-io/milvus-docker-compose/master/docker-compose.yml -o docker-compose.yml
   ```

3. 启动 Milvus 服务：

   ```bash
   docker-compose up -d
   ```

4. 检查 Milvus 服务状态：

   ```bash
   docker ps
   ```

   正常情况下会看到 milvus-standalone、etcd、minio 等相关容器处于运行状态。
5. （可选）部署 Attu（Milvus 图形化管理界面）：
   - Attu 是 Milvus 官方提供的 Web 可视化管理工具，便于查看、管理 Milvus 中的集合、数据、索引等。
   - 部署步骤：
     1. 在 Milvus docker-compose.yml 所在目录下，新增 Attu 服务配置：

        ```yaml
        attu:
          image: zilliz/attu:latest
          container_name: attu
          environment:
            - MILVUS_URL=milvus-standalone:19530
          ports:
            - "3000:3000"
          depends_on:
            - milvus-standalone
        ```

     2. 保存 docker-compose.yml 后，重新启动服务：

        ```bash
        docker-compose up -d
        ```

     3. 在浏览器访问 <http://localhost:3000> ，即可进入 Attu 管理界面。
     4. 登录时，Milvus 地址填写 milvus-standalone:19530，用户名和密码可留空（默认无认证）。
   - 通过 Attu 可进行集合管理、数据导入、向量检索、索引管理等操作，极大提升 Milvus 的可用性和可视化体验。
6. 注意事项：
   - 确保 Docker Desktop 已正常运行。
   - Milvus 默认使用 19530 端口（可在 docker-compose.yml 中修改）。
   - Attu 默认使用 3000 端口（如有冲突可修改）。
   - 启动后如需停止服务，可执行 `docker-compose down`。
   - 如遇端口冲突或资源不足，请调整 Docker 配置或释放端口资源。

### 2.3.1 Milvus 官方文档查阅

- Milvus 官方文档地址：<https://milvus.io/docs>
- 官方文档内容包括：
  - 安装与部署（单机、集群、云原生等多种方式）
  - 数据模型与集合管理
  - 向量检索、索引类型、参数优化
  - 与 Python/Java/Go 等多语言SDK集成
  - 常见问题与故障排查
- 推荐在遇到高级配置、集群部署、性能调优、API调用等问题时，直接访问上述官方文档获取权威帮助。

### 2.4 配置 Java 与 Maven

1. 安装 Java 17 及以上版本，并配置 JAVA_HOME 环境变量。
2. 安装 Maven 3.6 及以上版本，并配置 MAVEN_HOME 环境变量。
3. 在命令行输入 `java -version` 和 `mvn -v` 检查环境是否配置成功。

### 2.5 配置后端服务

1. 编辑 `src/main/resources/application.properties` 或 `application.yml`，填写如下关键信息：

   ```properties

   # DeepSeek API配置
   deepseek.api.url=https://api.deepseek.com/v1/chat/completions
   deepseek.api.key=your-deepseek-api-key
   # 文件存储路径
   file.upload.path=./uploads
   # 数据库配置（开发默认H2，可切换MySQL/PostgreSQL）

   ```

2. 如需连接 MySQL/PostgreSQL，请根据实际情况修改数据库相关配置。

### 2.6 启动后端服务

1. 在项目根目录下执行：

   ```bash
   mvn spring-boot:run
   ```

2. 启动成功后，后端服务默认监听 8080 端口。

### 2.7 访问前端页面

- 教师端：<http://localhost:8080/teacher.html>
- 学生端：<http://localhost:8080/student.html>
- 管理端：<http://localhost:8080/admin.html>
- 登录页：<http://localhost:8080/login.html>
- 主页：<http://localhost:8080/SmartEdu.html>

## 3. 登录与注册

- 访问 <http://localhost:8080/login.html>
- 新用户可点击“注册”，选择教师或学生角色，填写信息后完成注册。
- 管理员账号由系统预设或由超级管理员创建。
- 登录后系统会根据角色自动跳转到对应首页。

## 4. 教师端操作指南

### 4.1 课程管理

- 进入教师端主页
- 进入教师端主页（/teacher.html），点击“课程管理”。
- 可新建课程，填写课程名称、简介、学科等信息。
- 编辑、归档、删除已有课程。
- 管理课程成员，邀请学生加入课程。

### 4.2 资料上传与知识库

- 在课程详情页，点击“资料管理”或“上传资料”。
- 支持上传TXT、PDF、Word等格式。
- 上传后系统自动提取文本，构建知识库。
- 可查看、下载、删除已上传资料。

### 4.3 智能教案与题库

- 在课程详情页，点击“智能教案”生成教学大纲（可勾选联网搜索，以获取更广泛的即时资讯）。
- 系统基于知识库和AI自动生成教学目标、重点难点、思政设计等。
- 题库管理：选择章节/知识点、题型，点击“智能出题”，系统自动生成题目及解析（可勾选联网搜索，以获取更广泛的即时资讯）。
- 可手动编辑题目、解析和参考答案。

### 4.4 考试管理

- 点击“考试管理”，新建考试，设置考试名称、时间、班级、题型等。
- 选择题目（可从题库导入或AI生成）。
- 发布考试，考试到点自动开放。
- 考试结束后可选择是否公布答案与解析。
- 查看学生答卷、成绩，支持人工复查。

### 4.5 学情分析

- 在“学情分析”模块查看学生成绩分布、错题统计、知识点掌握情况。
- 系统自动生成教学改进建议（可勾选联网搜索，以获取更广泛的即时资讯）。
- 支持导出分析报告。

### 4.6 公告与消息

- 在“公告管理”发布课程通知、考试提醒等。
- 学生端会收到推送消息。
- 可查看历史公告、删除过期通知。

### 4.7 热点推送

- 在“热点推送”模块，教师可输入关键词或主题，系统自动从知识库和外部资讯中抓取相关热点。
- 教师可筛选、编辑热点内容，并一键推送给指定课程的学生。
- 推送后，学生端将收到实时提醒，促进对前沿知识的关注。

## 5. 学生端操作指南

### 5.1 选课与课程浏览

- 登录后进入学生端主页（/student.html）。
- 在“课程中心”浏览可选课程，点击“加入课程”参与学习。
- 可随时退出课程。

### 5.2 学习与资料下载

- 在“我的课程”中查看课程资料、公告。
- 支持下载教师上传的学习资料。

### 5.3 在线考试与作业

- 在“考试中心”查看待参加考试，点击进入答题。
- 支持自动保存进度，考试结束前可多次修改答案。
- 提交后可查看成绩和解析（如教师已公布）。
- 作业提交：在作业模块上传作业文件或在线作答。

### 5.4 成绩与学情分析

- 在“成绩中心”查看各科成绩、历史考试记录。
- 查看错题本、知识点掌握图谱，获取个性化学习建议。

### 5.5 AI学习助手

- 在“AI助手”模块输入问题，选择课程，系统基于知识库和大模型智能解答。
- 支持知识检索、疑难解答、学习建议等。

### 5.6 消息与互动

- 在“消息中心”接收课程公告、考试提醒、教师反馈等。
- 可与教师、同学进行互动交流。

### 5.7 热点推送

- 在“热点中心”或消息通知中，查看教师推送的学科前沿、行业动态等热点资讯。
- 可对热点内容进行收藏、评论，与其他同学展开讨论。
- 系统会根据学生的学习兴趣，个性化推荐相关热点。

## 6. 管理端操作指南

### 6.1 用户管理

- 登录管理端（/admin.html），进入“用户管理”。
- 支持批量导入、创建、编辑、删除教师和学生账号。
- 分配角色、重置密码、禁用/启用账号。

### 6.2 课程与考试全局管理

- 审核课程、监控考试进度。
- 管理所有课程资料、题库、考试数据。
- 支持全局搜索、导出数据。

### 6.3 合规与AI安全

- 在“合规管理”上传、维护政策文档。
- 监控AI内容合规，检测敏感内容、异常行为。
- 生成合规报告，支持审计与追踪。

### 6.4 系统统计

- 查看用户活跃度、课程参与度、考试通过率等多维数据统计。
- 支持导出统计报表。

## 7. 常见问题与解答

- **Q: 忘记密码怎么办？**
  A: 可在登录页通过“忘记密码”找回，或联系管理员重置。
- **Q: 文件上传失败？**
  A: 检查文件大小、格式是否支持，或网络连接。
- **Q: AI助手无法使用？**
  A: 检查系统是否已配置有效的AI API密钥，Milvus服务是否正常运行。
- **Q: 无法加入课程/考试？**
  A: 请确认课程状态、考试时间，或联系教师/管理员。
- **Q: 页面显示异常？**
  A: 建议使用最新版 Chrome/Edge 浏览器，清理缓存后重试。

## 8. 注意事项

- 请勿上传涉密、违规或侵权内容。
- 文件上传、AI调用等操作需保证网络畅通。
- 管理员应定期备份数据，监控系统运行状态。
- 生产环境建议使用MySQL/PostgreSQL数据库。
- 合规政策、AI安全检测可根据实际需求调整。
